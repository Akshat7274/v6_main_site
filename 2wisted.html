  • 2Twisted

In this event, there is a buzzer round quiz for which limited time will be allotted. It will consist of survey based questions, fill in the blanks, MCQs, match the following and arrange in the sequence.

Requirements

  1. Classroom to conduct the sessions/competition
  2. People are required for conducting events
  3. Projector
  4. Blank sheets
  5. Technical team for any error management

Procedure

  1. Round 1: It would be buzzer based. The contestant who sounds the buzzer fastest and answers the question correctly will win the round.
  2. Round 2: It would be a survey-based question and the contestant fastest to answer the question would win.
  3. Round 3: It will be a fill-in-the-blanks round.
  4. Round 4: It would be a match the following round.
  5. Round 5: A particular sequence would be given which needs to arrange in a proper sequence or chronology.
  6. Round 6: It would be a multiple-choice question round and there would be more than 1 correct answer with 6 - 7 options

Round Format

  1. Round 1 - Multimedia Round (Images/Audio/Video-Based Question)
  2. Round 2 - Survey Based Multiple Choice Question
  3. Round 3 - Pure Text-Based Questions (Direct 1 - 1 Questions)
  4. Round 4 (2wisted Mode) - Dice roll
  • Set 1 = Who am I?
  • Set 2 = True/False
  • Set 3 = Fill in the blanks
  • Set 4 = Match the Following
  • Set 5 = Arrange the given option in correct sequence/ chronology
  • Set 6 = Multiple MCQs questions (6-7 options)
